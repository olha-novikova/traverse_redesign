Stable tag: 1.1.0

Add an "Apply with XING" button to job listings which have an 'email' apply method. Requires API keys from XING (https://dev.xing.com/).

= Documentation =

Usage instructions for this plugin can be found on the documentation site: [https://wpjobmanager.com/documentation/add-ons/apply-with-xing](https://wpjobmanager.com/documentation/add-ons/apply-with-xing).

== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)
